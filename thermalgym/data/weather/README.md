# EPW Weather Files

These EnergyPlus weather files are required for ThermalGym simulation but are not included in the repository due to size.

## Required Files

- USA_MN_Duluth.726550_TMY3.epw  (cold climate / Zone 6)
- USA_TX_Houston-Bush.Intercontinental.AP.722430_TMY3.epw  (hot climate / Zone 2)
- USA_OH_Columbus.Intl.AP.724280_TMY3.epw  (mixed climate / Zone 4)

## Download

Download from the EnergyPlus weather database: https://energyplus.net/weather
