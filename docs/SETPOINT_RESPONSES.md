# Setpoint Response Dataset

## Overview

This dataset captures HVAC system responses to setpoint changes. Each episode starts when a setpoint changes by >1°F and ends when indoor temperature reaches the new target.

**Data quality**: Episodes are filtered to only include cases where the target was reached without the setpoint changing again mid-episode.

| Attribute | Value |
|-----------|-------|
| File | `data/setpoint_responses.parquet` |
| Size | 72 MB (compressed) |
| Rows | 1.9M |
| Episodes | 158,604 |
| Generated by | `scripts/extract_setpoint_responses.py` |

---

## Episode Types

### Heat Increase
- **Trigger**: Heat setpoint increases by >1°F AND indoor temp is below new setpoint
- **End condition**: Indoor temp >= new heat setpoint
- **Count**: 101,985 episodes (64%)

### Cool Decrease
- **Trigger**: Cool setpoint decreases by >1°F AND indoor temp is above new setpoint
- **End condition**: Indoor temp <= new cool setpoint
- **Count**: 56,619 episodes (36%)

## Usage

### Load Data

```python
import pandas as pd

df = pd.read_parquet("data/setpoint_responses.parquet")

# Filter by type
heat_episodes = df[df["change_type"] == "heat_increase"]
cool_episodes = df[df["change_type"] == "cool_decrease"]

# Filter by split
train = df[df["split"] == "train"]
```

### Get Episode Trajectories

```python
# Get unique episodes
episode_ids = df["episode_id"].unique()

# Extract single episode trajectory
ep = df[df["episode_id"] == episode_ids[0]].sort_values("timestep_idx")

# Temperature over time
temps = ep["Indoor_AverageTemperature"].values
target = ep["target_setpoint"].iloc[0]
```

### Compute Time to Target

```python
# Episode durations (in timesteps)
durations = df.groupby("episode_id").size()

# Convert to minutes
durations_min = durations * 5
```

## Data Quality

Episodes are filtered to ensure clean training data:

1. **Target reached**: Only episodes where indoor temp reached the setpoint
2. **No mid-episode setpoint changes**: Episodes end early if setpoint changes by >0.5°F
3. **No missing data**: Episodes end if sensor data gaps occur

This filtering removed ~24% of episodes and ~84% of rows compared to unfiltered extraction.

---

## Use Cases

1. **Predict time to reach setpoint** - Given initial conditions, how long until target temp?

2. **Predict temperature trajectory** - Model the temp curve during setpoint response

3. **MPC training data** - Learn system dynamics for model predictive control

4. **Precooling/preheating optimization** - When to start HVAC to reach target by deadline?
